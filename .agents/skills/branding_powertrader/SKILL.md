# Skill: Estilo y Marca - PowerTrader

Marca: PowerTrader (y ecosistema: Energyflash, Energyvision, PowerRadar)

## Cuándo usar este Skill
- Si vas a diseñar una UI o una pantalla para PowerTrader o sus sub-marcas.
- Si vas a crear una landing page o artículos en Hugo (PaperMod).
- Si vas a escribir textos: titulares, CTAs, hilos de X o correos.
- Si vas a generar activos visuales o diagramas.

## Regla número 1
No improvises el estilo. Consulta siempre los recursos en `recursos/` para mantener la coherencia técnica y visual.

## Dónde mirar según el tipo de tarea
- **Estilo visual** (colores, tipografías, espaciado): `recursos/estilo-visual.json`
- **Forma de escribir** (tono, estructura, vocabulario): `recursos/guia-de-textos.md`
- **Logos y Activos**: `recursos/logos/` (Usar SVGs para máxima calidad).

## Checklist antes de entregar
1. ¿El diseño respeta el modo oscuro (PaperMod Dark)?
2. ¿Se han usado los emojis correctos (⚡ para Vision, ☀️ para Solar, 🔬 para Tech, 💡 para Ahorro)?
3. ¿El tono es experto pero directo y accesible?
4. ¿Se ha incluido el logo correcto de la sub-marca si aplica?

## Cómo mejorar este Skill
Ajusta los archivos JSON o MD en la carpeta `recursos/` basándote en el feedback del usuario.
