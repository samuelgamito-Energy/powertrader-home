# Guía de textos (tono y estilo) - Alumbra

## Cómo suena la marca
Sincera, humana y cercana. Alumbra busca transmitir el calor y la cercanía de una llama ("Alumbra"). El lenguaje debe ser claro, directo y con "actitud" (proactiva y positiva). Es un tono experto pero accesible, evitando la opacidad tradicional del sector energético.

## Objetivo
Transmitir confianza y transparencia, posicionando a Alumbra como el socio ideal para la transición ecológica y el ahorro energético.

## Valores y Pilares
- **Actitud**: Proactividad y positividad en el servicio.
- **Transparencia**: Claridad total en tarifas y procesos.
- **Sostenibilidad**: Energía 100% verde y compromiso con la descarbonización.
- **Honestidad**: Comunicación sincera y sin costes ocultos.

## Reglas de Escritura e Identidad
1. **Tipografía**: Usa **Aptos** para documentos de texto y **Avenir Next** para presentaciones. Si no están disponibles, usa Arial.
2. **Jerarquía Visual**:
   - Usa **12pt** para fragmentos destacados o introducciones.
   - Usa **10pt** para textos corridos y enumeraciones.
3. **Uso de Color**:
   - Aplica el morado **#4F1E50** para textos destacados y títulos principales.
   - Aplica el rosa **#CE326B** para CTAs, acentos o elementos dinámicos.
   - Usa el gris **#595959** para contraste secundario.
4. **Claridad sobre todo**: Explica los conceptos energéticos de forma que cualquier cliente los entienda.
5. **Cercanía**: Usa un lenguaje humano, no excesivamente burocrático, pero manteniendo el rigor de una figura de Director General.

## Palabras que sí
Actitud, energía verde, transición ecológica, transparencia, cercanía, vector energético, descarbonización.

## Palabras que no
Costes ocultos, opacidad, combustibles fósiles, gris.

## Eslóganes de referencia
- "Energía verde y con actitud."
