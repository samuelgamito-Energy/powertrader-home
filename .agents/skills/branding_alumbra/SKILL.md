# Skill: Estilo y Marca - Alumbra

Marca: Alumbra ("Energía verde y con actitud")

## Cuándo usar este Skill
- Si vas a diseñar una UI o una pantalla para Alumbra.
- Si vas a crear contenido corporativo o estratégico para el Director General.
- Si vas a escribir textos oficiales o comunicados de marca enfocados en sostenibilidad y transparencia.

## Regla número 1
El tono debe ser **sincero, humano y con actitud**. Consulta siempre `recursos/guia-de-textos.md` para asegurar que la comunicación no parezca gris u opaca.

## Dónde mirar según el tipo de tarea
- **Estilo visual**: `recursos/estilo-visual.json`
- **Forma de escribir**: `recursos/guia-de-textos.md`
