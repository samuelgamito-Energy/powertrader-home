# Ecosistema PowerTrader

El proyecto se articula a través de varios productos y canales vinculados por un mismo núcleo de datos.

## Plataformas Principales
- **Web**: powertrader.es (Centro neurálgico).
- **Redes Sociales**: Foco en X y LinkedIn.
- **Newsletter**: Envío diario con el resumen de mercado.

## Sub-marcas y Secciones
| Marca | Frecuencia | Foco |
| :--- | :--- | :--- |
| **EnergyFlash** | Diaria | Informe rápido de mercado. |
| **PowerRadar** | Semanal | Informe profundo, análisis de fuentes automáticas y podcast. |
| **EnergyVision** | Mensual | Análisis de tendencias a largo plazo. |
| **PowerAhorro** | Bajo demanda | Consejos prácticos de ahorro para el sector residencial. |
| **EnergyTech** | Bajo demanda | Innovación: BESS (baterías), autoconsumo, puntos de recarga. |
| **ProductoSolar** | Bajo demanda | Especializado en generación fotovoltaica (generadores). |
